Pueblo HTML extensions

Most of Pueblo's HTML extensions are prefixed with xch_ ('x' for eXtension, 'ch' for Chaco), to avoid conflicts with standard HTML and with other extensions. Chaco’s intent was to try to get some of these features included in future versions of the HTML standard. This may have been accomplished, but at this point the elements have been left using the original names, for compatibility.
